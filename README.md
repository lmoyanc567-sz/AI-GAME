# 学生成绩分析与可视化系统

## 系统简介

这是一个功能完整的学生成绩分析与可视化系统，旨在帮助教育工作者直观地分析学生成绩数据，识别学习模式，并提供个性化的学习建议。系统通过"小鸭子游泳比赛"的生动隐喻，将复杂的学习能力评估转化为直观有趣的视觉体验。

## 核心功能

1. **数据导入与解析**：支持CSV和Excel格式的成绩数据文件
2. **小鸭子游泳比赛可视化**：将学生学习能力转化为直观的动画
3. **多维数据分析面板**：提供班级整体分析和学生个体分析
4. **智能学习能力评估**：综合考虑成绩水平、稳定性和进步趋势
5. **个性化学习建议**：基于数据分析生成针对性的学习建议
6. **分析报告导出**：一键生成并下载班级分析报告

## 数据模板使用说明

### CSV模板

系统提供了`成绩数据模板.csv`文件作为参考。您可以使用此模板填写自己的学生成绩数据，或按照以下格式创建新的CSV文件：

**必填字段**：
- 姓名：学生姓名
- 总分：学生所有科目的总分
- 排名：学生在班级中的排名

**可选字段**：
- 语文、数学、英语、物理、化学、生物等：各学科成绩
- 历史成绩：可在系统中手动添加，用于计算成绩稳定性和进步趋势

### 如何使用模板

1. 使用Excel或其他电子表格软件打开`成绩数据模板.csv`
2. 替换模板中的示例数据为您的实际学生成绩数据
3. 确保保留原始的列名和格式
4. 保存文件（保持CSV格式）
5. 在系统中上传您的成绩数据文件

## 系统使用指南

1. **数据导入**：
   - 点击"浏览文件"按钮选择您的成绩数据文件（CSV或Excel格式）
   - 或直接拖放文件到上传区域
   - 也可以点击"使用演示数据"按钮加载示例数据

2. **小鸭子游泳比赛**：
   - 点击"开始比赛"按钮启动动画
   - 点击任意小鸭子查看该学生的详细分析
   - 使用"重置"按钮重新开始动画

3. **数据分析**：
   - 在"班级整体分析"标签页查看班级整体情况
   - 在"学生个体分析"标签页查看选定学生的详细分析
   - 使用表格排序功能按不同维度排序学生数据

4. **报告导出**：
   - 点击"导出报告"按钮生成班级分析报告
   - 可以复制报告内容或下载为文本文件

## 技术说明
